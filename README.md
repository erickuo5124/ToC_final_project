# Line Bot - 小測驗

![](https://i.imgur.com/BQiOwIf.png)

## 功能
- 對使用者進行小測驗，並得到回覆的答案
- 串接 google api，直接從 google 試算表動態更改題目以及得到回覆的答案

## 使用方法
### Line 機器人
1. 輸入"開始測驗"

![](https://i.imgur.com/qtMQB7r.png)

2. 輸入名字
