# Line Bot - 小測驗

![](https://qr-official.line.me/sid/L/128rsqyx.png =300x)

## 功能
- 對使用者進行小測驗，並得到回覆的答案
- 串接 google api，直接從 google 試算表動態更改題目以及得到回覆的答案
